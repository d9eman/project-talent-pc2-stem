# PC1, PC2, and STEM Pathways in Project TALENT

This repository contains the final code, aggregate result tables, and figures my Advanced Data Science project extending my thesis on childhood gender nonconformity and STEM pathways in Project TALENT.

## Research question

The original thesis used PC1 from a five-domain PCA to construct a Gender Conformity Index (GCI) and classify childhood gender nonconformity (CGN). This project asks what PC2 captures and whether it helps explain which CGN students move toward or away from STEM.

## Main finding

PC1 captures broad gender conformity/nonconformity. PC2 captures a secondary contrast between technical/physical interests and social/domestic/expressive interests. CGN girls with high PC2 show the strongest early STEM and engineering interest. PC2 is much weaker for later STEM major and job outcomes.
